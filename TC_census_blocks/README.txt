The file "TC_census_blocks.kmz" contains the US Census blocks for Travis County.
(A KMZ file is a KML file enclosed in a ZIP archive.)

This dataset was produced from:
http://www2.census.gov/geo/tiger/TIGER2013/TABBLOCK/tl_2013_48_tabblock.zip

It was generated by extracting the Travis County census blocks (COUNTYFP == 453)
and saving as a KML layer.
