This site contains data published by Open Austin prior to 2015. More
recent datasets can be found on Open Austin's page on data.world:

https://data.world/openaustin

Typically this data is derived from government information, converted
or processed to some form more amenable for public use.

The source to this repository is here:

https://github.com/open-austin/data-open-austin-org

If you are a github user, you can open issues or submit changes (pull
requests) for publication.
