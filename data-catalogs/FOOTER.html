The files in this directory are dumps of the metadata for datasets published at data.austintexas.gov

Each file is a JSON object with elements:

  • count: number of data catalogs
  • searchType: "views"
  • timestamp: time at which pull occurred, integer seconds since epoch
  • results: array of catalog metadata, one entry per dataset

The dumper tool is here: https://github.com/chip-rosenthal/pull-data-portal-catalog.
